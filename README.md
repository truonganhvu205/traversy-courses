# Clone project
```bash
git init
git clone https://github.com/truonganhvu205/traversy-courses.git
cd traversy-courses
npm install
```

# Libraries
## React Icons
```bash
npm install react-icons --save
```

## uuid
```bash
npm install uuid
```

# Getting Started
```bash
npm run dev
```
